# 📦 mdlv (Markdown Layered Viewer)
> **The AI-Native Markdown Viewer for the Age of Deep Reasoning.**

`mdlv` is a high-performance macOS desktop application built with **Tauri v2** and **Rust**. It is designed to navigate the long, deeply nested, and complex Markdown outputs generated by modern LLMs.

## 🚀 Why mdlv?
In the era of AI, we no longer just read documents; we navigate through complex reasoning chains. Traditional viewers are too linear for the 5+ levels of nesting that AI often produces. 

**mdlv** treats Markdown as a **tree (AST)**, not just text.

### Key Features
- **Hierarchical Focus:** Automatically collapse deep sub-sections to stay focused on high-level logic.
- **Breadcrumb Navigation:** `H1 > H2 > H3` sticky headers to prevent losing context.
- **Instant Structural TOC:** A lightning-fast sidebar powered by Rust-based parsing.
- **Live Watcher:** Instantly reflects changes when an AI agent or script updates local files.

## 🛠 Tech Stack
- **Core:** Rust (Tauri v2)
- **UI:** React + TypeScript + Tailwind CSS
- **Parser:** AST-based structural analysis in Rust (`pulldown-cmark`)

## 💻 Getting Started

### Prerequisites
- Node.js (v20+)
- Rust (stable, installed via `rustup`)

### Installation & Run
```bash
# Clone the repository
git clone https://github.com/nkchan/mdlv.git
cd mdlv

# Install dependencies
npm install

# Start the development server and desktop app
npm run tauri dev
```

### 🚨 For Developers
Before pushing changes (especially UI or structural changes), please run the production build locally to ensure everything compiles correctly:
```bash
npm run tauri build
```
Verification of the production build is required to avoid breaking the CI/CD pipeline.
